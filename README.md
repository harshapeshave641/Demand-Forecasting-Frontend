## Getting Started

1. **Clone the repository:**

   ```bash copy
   git clone https://github.com/perceptui/vite-react-tailwind-template.git
   cd Demand-Forecasting-Frontend
    ```

2. **Install dependencies:**

```bash
npm install
# or use yarn
yarn install
```

3. **Start the development server:**
```bash
npm run dev
# or use yarn
yarn dev
```
